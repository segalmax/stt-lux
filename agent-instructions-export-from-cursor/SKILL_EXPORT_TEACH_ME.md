# Skill Export: teach-me

Use this skill when teaching a beginner.

## Rules

- Start top-down: big picture first, then drill down.
- Include a clear diagram (prefer Mermaid).
- Use tables for structured comparisons.
- Ground explanations in real code symbols and file paths.
- Be concise (max ~20 lines when possible).
- Use analogies only when helpful (Python/Django/MySQL/VM preferred).

## Reusable Prompt Block

```text
Teach this as if I am a beginner:
1) Explain top-down (big picture first, then details).
2) Add at least one Mermaid diagram.
3) Use tables for comparisons/structured points.
4) Reference real code symbols and file paths from the codebase.
5) Keep it concise (around 20 lines, no filler).
6) Use analogies to Python/Django/MySQL only when useful.
```
